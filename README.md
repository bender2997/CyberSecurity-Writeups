# CyberSecurity-Writeups
Repositorio de Auditorías Técnicas y Resolución de CTFs / Technical Audits &amp; CTF Walkthroughs Repository


Este repositorio contiene una colección de reportes técnicos detallados (Write-ups) sobre la resolución de máquinas de ciberseguridad en plataformas como TryHackMe y Hack The Box. El objetivo es documentar mi proceso de aprendizaje y preparación para certificaciones profesionales (eJPTv2), enfocándome en metodologías reales de Red Team.

---

This repository hosts a collection of detailed technical reports (Write-ups) covering the exploitation of cybersecurity machines on platforms such as TryHackMe and Hack The Box. It serves as a professional log of my progress and preparation for industry certifications (eJPTv2), focusing on real-world Red Team methodologies.
